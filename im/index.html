One-time setup needed

This page talks directly to the Google Sheets API from your browser — no server involved. Before it can sign you in, you need to create a Google OAuth client and paste its ID into this file.

  1. In Google Cloud Console, create a project (or reuse one).
  2. Enable the Google Sheets API for that project (APIs & Services → Library).
  3. Go to APIs & Services → OAuth consent screen. Choose External, keep it in Testing status, and add gzarruk@gmail.com as a test user.
  4. Go to APIs & Services → Credentials → Create Credentials → OAuth client ID → type Web application. Under Authorized JavaScript origins add https://www.gzarruk.com (and http://localhost:PORT if you want to test locally).
  5. Copy the generated Client ID and paste it as CONFIG.clientId near the top of the <script> in im/index.html.
