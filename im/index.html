One-time setup needed

This page reads directly from Cloud Firestore using the Firebase JS SDK — no server involved. Before it can sign you in, you need a Firebase project with Firestore and Google sign-in enabled.

  1. In Firebase console, create (or reuse) a project.
  2. Enable a Cloud Firestore database (Native mode, default database, production rules).
  3. Under Authentication → Sign-in method, enable the Google provider, and add your site's domain under Authorized domains.
  4. Lock down Firestore security rules so only your account can read/write the trainingPlan collection.
  5. Register a Web app (Project settings → Your apps) and paste its config object as CONFIG.firebase near the top of the <script> in im/index.html.
